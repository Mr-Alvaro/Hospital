### Descripcion

Este proyecto consiste en la seguridad del Hospital Nacional Cayetano Heredia. Se incluye funcionalidades como:

- Gestionar la seguridad de la aplicación.

<div id='dependencias'/>

### Dependencias
- mssql-jdbc-10.2.1.jre8
- swagger-annotations-1.6.3
- lombok-1.18.26
- spring-boot-starter-web-2.7.10
- jackson-databind-nullable-0.2.3
- spring-boot-starter-security-3.1.2

----------
<div id='requisitospreviosparaejecutarlaaplicación'/>

### Pre-requisitos para ejecutar la aplicación

Este proyecto requiere la previa instalacion de IntelliJ IDEA 2023.1.3 o alguna otra IDEA que usen, tambien necesitaran la base de datos SQLServer y Posmant para poder hacer las consultas al proyecto.

----------
<div id='instalacion'/>

### Instalacion

- Clona este repositorio utilizando https://github.com/InformaticaHNCH2/ms-seguridad.git
- Dirigete a la rama /develop
- Tener instalada la versión Java 17 y configurada en las variables de entorno
- Descargar dependencias (Maven)
- Ejecutar el proyecto

----------
<div id='construidocon'/>

### Construido con
- **_Java 17 (spring boot 2.7.3.RELEASE)_** - lenguaje base de toda la aplicación
- **_Maven (version 3.6.3)_** - Gestor de dependencias
- **_Microsoft SQL Server 2022 (version 16.0.1000.6)_** - Base de datos relacional de la aplicación.
- **_Postman (version 10.18.9)_** - Gestor de dependencias
